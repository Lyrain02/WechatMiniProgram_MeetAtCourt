# 有球必应——微信小程序 前端
  本项目是一个大学生双创培育项目，主要实现了微信小程序的前端开发。
  有球必应是一款致力于解决高校体育社团管理问题的微信小程序，其主要功能有：发起活动、查询活动、加入活动。
  
### 实现
  基于微信小程序的.js/.json/.wxss/.wxml
